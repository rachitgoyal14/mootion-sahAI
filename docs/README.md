# Mootion Backend Docs

This folder documents the current backend design and implementation state.

## Contents

- `BACKEND_OVERVIEW.md` - architecture, runtime model, and code boundaries
- `AUTH_AND_RBAC.md` - login, OAuth, sessions, and role guards
- `ONBOARDING_AND_CLASSES.md` - teacher/student onboarding and class membership flows
- `CURRICULUM_PATCHING.md` - roadmap shape, versioning, and patch operations
- `CURRICULUM_MANUAL_FLOW.md` - manual curriculum creation and NCERT bootstrap flow
- `CHAPTERS.md` - chapter bootstrap, placeholder assets, and why chapters exist
- `ASSIGNMENTS.md` - assignment-driven generation flow and queue behavior
- `MEDIA_AND_WORKERS.md` - Redis job queue, MinIO storage, and worker runtime
- `API_REFERENCE.md` - endpoint catalog and practical usage map
